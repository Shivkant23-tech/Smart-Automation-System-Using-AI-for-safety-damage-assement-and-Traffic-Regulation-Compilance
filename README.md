# Smart-Automation-System-Using-AI-for-safety-damage-assement-and-Traffic-Regulation-Compilance
_This project present an advance smart automative system that Artifical Intelligence, Machine Learning, Deep Learning, Nartural Languages Programing and Internet of Thinks technologies to enchance vehicle safety security, and operational intelligence. The system is designed to monitor analayse and ensure compliance with traffic regulations, while providing the real time insigth into vehicle conditions and driver behaviour, effectively capturing the essesence of intelligence and responsible transporations.
_The system integrated multiple advance feactures, including :
  1. Vehicle Damage Detection and severity estimation using Deep Learning model (CNN,YOLO).
  2. Seatbelt detection system to ensure passenger safety compliance.
  3.  Passenger | Student Detection to prevent overloading.
  4.  Fuel Monitoring System Using IoT sensor.
  5.  Traffic Rule Compliance Detection (Signals, Speed limit, lane discipline).
  6.  Driver Behaviour Analysis for identify unsafe driving pattern.
  7.  AI Chatbot assitant (NLP-based) for real-time alerts, report and interaction.
  8.  Vehicle Security Monitoring to detect anomalies and unauthorized access.

This system utlize IoT-enabled sensor for real-time data collection such as fuel level speed, and enviroment conditions. AI-driven computer vision technquies ensure accurate detection and monitoring which cloud integration provides servers storages, scalability and remote access.

**Advantages Machine Learning Algorithm enables**
   a. Predictive Maintance to detect potential failures early.
   b. Anomaly Detection for unsual system behaviour.
   c. Data Driven insights to improve decision making over time.
